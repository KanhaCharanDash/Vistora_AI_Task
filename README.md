# MERN Shop (Frontend)

A simple e-commerce frontend built with **React**, **Material UI**, **Zustand**, and **React Router**.

---

## Features
- Product listing
- Add to cart with quantity control
- Wishlist (liked items)
- Cart & wishlist persist after refresh
- Dynamic cart total calculation
- Responsive UI
- Sticky footer & reusable layout

---

## Tech Stack
- React
- Material UI (MUI)
- Zustand
- React Router DOM
- JavaScript

---

## Requirements
- Node.js (v16 or higher)
- npm or yarn

Check:
```bash
node -v
npm -v
